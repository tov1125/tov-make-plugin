# Slide-HDML

순수 HTML/CSS/JS 단일 파일로 완결되는 웹 슬라이드 프레젠테이션을 생성하는 Claude Code 플러그인.

## 개요

외부 라이브러리 없이 브라우저에서 바로 열면 동작하는 16:9 슬라이드를 생성한다.
자연어 요청 또는 기존 문서(MD, PDF, URL, PPTX, HWPX 등)를 슬라이드로 변환할 수 있다.

## 주요 기능

- **자연어 → 슬라이드**: "발표자료 만들어줘"로 바로 생성
- **문서 변환**: Markdown, 텍스트, PDF, URL, PPTX, HWPX → HTML 슬라이드
- **전환 효과**: fade, slide, zoom 3종 전환
- **빌드 애니메이션**: fade-up, grow-in 등 순차 등장 효과
- **코드 하이라이트**: 라인별 하이라이트 + 단계별 진행
- **2단 컬럼 레이아웃**: 비교/대조 슬라이드
- **발표자 노트**: S키로 토글
- **터치 스와이프**: 모바일 지원
- **키보드/풀스크린**: 화살표키, F키, 진행바

## 테마

| 테마 | 용도 |
|------|------|
| `theme-dark` | 타이틀, 마무리 슬라이드 |
| `theme-light` | 본문 슬라이드 |
| `theme-accent` | 강조, 핵심 메시지 |
| `theme-code` | 코드 블록 슬라이드 |

## 워크플로우

```
입력 분석 → 슬라이드 계획 → HTML 생성
```

1. **입력 분석**: 자연어 또는 문서에서 콘텐츠 추출
2. **슬라이드 계획**: 구성안을 사용자에게 제시
3. **HTML 생성**: 내장 템플릿의 슬라이드 영역만 교체하여 단일 파일 생성
